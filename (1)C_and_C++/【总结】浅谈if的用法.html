【总结】浅谈 if 的用法

“if ... else ...”语句是我们遇到最简单的一种程序控制语句。 它只有两个分支，所以在实际使用的过程中基本不会有什么问题。 但是如果遇到多路分支的时候，如 if ... else if ... else if ... ... else ... 如果理解不到位，有时候就会有一点问题了。作者之前就遇到了一些问题，这里整理出来大家一起学习。

数据类型总结

整型变量

·        短整型（2B = 16bit）

o   有符号短整型：[signed] short [int]

o   无符号短整型：unsigned short [int]

·        基本整型（4B = 32bit）

o   有符号基本整型：[signed] int

o   无符号基本整型：unsigned int

·        长整型（4B = 32bit）

o   有符号长整型：[signed] long [int]

o   无符号长整型：unsigned long [int]

·        注意：

o   整型变量的溢出：如果变量的数值超过该类型变量的最大范围，则发生数据溢出，但程序不会报错！！所以要十分小心此类错误。

 

实型变量

·        单精度实型变量：float（4B = 32bit）

·        双精度实型变量：double（8B = 64bit）

·        长精度实型变量：long double（8B = 64bit）

 

字符型变量

·        char

·        实质存储的是ASCLL码，所以字符变量可以和整型数据通用，还可以和整型数据运算

前 言

简 介

Python是一种跨平台的计算机程序设计语言。 是一种面向对象的动态类型语言， 最初被设计用于编写自动化脚本(shell)， 随着版本的不断更新和语言新功能的添加，越来越多被用于独立的、大型项目的开发。

单词python是蟒蛇的意思，仔细看看图标就会发现是两条蛇

应 用

其实要用那些专业的术语解释Python是干什么的的确很晦涩难懂。如果你听过“爬虫” （如果不了解没关系，只要想想12306为什么要弄那么变态的验证码，你就理解“爬虫”有多么厉害， 当然爬虫也分好坏，搜索引擎本身就是爬虫）， 其背后的编写语言大多用的是Python。而Python还可以实现自动收发邮件，自动排版Word，进行数据库管理，网站管理等等。 这些应用场景听起来似乎毫不相关，但事实上这是Python有丰富的第三方库的原因。 可以简单理解第三方库为别人写好的一些现成代码，比如有人想自动收发邮件，就写了一些代码， 然后进行封装，发布在网上，你下载了直接用其中的一些代码就可以了。也就是大家说的“不要重复造轮子”。

安 装

由于Windows系统不能直接运行Python代码，所以需要安装一个“解释器”（也就是将Python代码解释为Windows系统下能直接执行的二进制代码）。 点击下方官网链接下载编译器，安装时一定要勾选“将路径添加到环境”。

下 载： Python 3.8

如果你只需要一个轻量化的代码编辑器（也称文本编辑器），推荐：Visual Studio Code
不是那个大出天际的Visual Studio，是微软推出的轻量化新编辑器，由于是轻量化的，所以需要手动配置： 下载编译器，下载对应语言的“扩展”，然后配置环境（仔细阅读帮助文档即可，软件内也有相应的提示信息）

下 载： Visual Studio Code （完全免费）

对于Python语言，还有一个热度很高的代码编辑器：Pycharm
由于没有深度使用，也不能给出更多的测评建议。但是的确和Visual Studio Code很像，也是轻量化的一款编辑器。 不过如果你想使用一些高级功能，需要付费的Professional版本，当然免费版作为初学是完全够用的。

下 载： Pycharm （下载免费的Community版本即可）

如果你想要完整的编程体验，那么你需要一款 IDE（Integrated Development Environment，集成开发环境）， 包括代码编辑器、编译器、调试器和图形用户界面等工具。 集成了代码编写功能、分析功能、编译功能、调试功能等一体化的开发软件服务套。
推荐：Visual Studio（大出天际的那个，不过现在有所改善），是微软成熟的开发软件，安装时选择需要的“工作负载”，无需手动配置。最新版本是2019。

下 载： Visual Studio 2019（下载免费的Community版本即可）

当然还有很多编辑器，大家可以根据自己的情况和喜好自行选择。
